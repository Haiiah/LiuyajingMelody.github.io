基于Tensorflow的深度学习平台CPU版本搭建

发表于 2019-04-26 | 分类于 深度学习

说明：电脑：win10（64位）。显卡属于Intel系列，不支持CUDA所以无法安装GPU版本的TensorFlow，只能退而求其次安装据说功能等同于GPU版速度却很慢的CPU版本。

一、安装anaconda

参考链接：https://www.anaconda.com/download/>

（1）下载anaconda安装包，下载地址： https://www.anaconda.com/download/#windows

（2）执行下载后的exe文件，安装。

注意安装anaconda时需要把环境变量加入windows环境中。这里建议安装完后手动加入，在最安装过程中，最好不要在第一个框里面打钩。 （反复安装了不下5遍，最后问题出在这里，血的教训，不过也有打钩安装成功的，这里注意一下，每个人的问题都不一样）。

（3）环境变量设置

因为我的环境变量是手动设置的，所以这里需要对环境变量进行设置，步骤如下：

首先找到anaconda的安装路径，如下图所示：

然后找到一个scripts文件，点击后将路径复制

然后，右键“我的电脑”>选择“属性”>高级系统设置>环境变量,然后，在系统变量中找到path

选中path ，然后点击编辑，将光标移到最后面，将键盘转换为英文输入法，输入一个分号 ( ; )，然后将复制的路径，粘贴在分号后面，点击确定，保存后关闭就可以，启动anaconda就可以使用了。

如果正确安装，从电脑开始那里搜索，可以得到

（4）检查安装结果。运行开始菜单—>Anaconda3—>Anaconda Prompt

检测anaconda环境是否安装成功：conda –version ，会得到conda的版本信息

检测目前安装了哪些环境变量：conda info –envs

二、安装tensorflow

对于现有的深度学习框架，现在使用最多的大概为基于符号式编程和命令式编程的下面两种框架。

符号式编程：最主流的代表，当然就是TensorFlow了。

命令式编程：最主流主流的代表就是PyTorch。

这里详细介绍tensorflow的安装步骤。

（1）进入windows命令模式，创建tfenv环境，安装python3.5

现有的tensorflow框架只支持python3.5和python3.6，而相比较而言，tensorflow更加稳定，所以这里安装3.5版本。其中tfenv是申请的环境名称，这里还可以设置为其他名称。

三、遇到的问题

（1）安装anaconda失败

首先检查你的环境变量是否设置，安装到百分之80的时候会跳出一个exe文件，切记不要关掉。

（2）anaconda prompt打开，所有的命令都不管用，报错，请注意使用管理员的身份运行
